WordPress Live Ajax Search

== Description ==

Searchlive is a live search plugin for WordPress. This live search engine provides a user friendly and modern ajax powered search form.


= PLUGIN FEATURES: =

* Search in posts and pages
* Search in title, description, excerpt and categories
* Automatic search replacement
* Fully ajax powered

= GDPR AND PRIVACY =
Searchlive Lite plugin is GDPR compliant:

* We don’t store any visitor or user data
* We don’t send any data to remote servers
* Our plugin is only used to search within website
